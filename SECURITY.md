# Security Policy

This is a casual, single-maintainer project.

## Reporting a Vulnerability

Please open an issue with enough detail to reproduce the problem, but do **not** post sensitive data (tokens, credentials, private logs) publicly.

If you want private handling first, mention that in a minimal public issue and I will provide a private contact path in follow-up.

## Supported Versions

Security fixes are provided on a best-effort basis for the current `main` branch only.
